    目前版本支援的AI模型和收費方式如下

    # Cohere(Command R) GPT-3.5 Turbo GPT-4 Turbo
    收費方式 Trial key：免費

    Production key：流量計費
    Input: $0.0005 / 1K TOKENS
    Output: $0.0015 / 1K TOKENS     		僅API流量計費，無需月費
    Input: $0.0005 / 1K tokens
    Output: $0.0015 / 1K tokens     		月費 + API流量計費
    Input: $0.01 / 1K tokens
    Output: $0.03 / 1K tokens
    使用限制 免費：
    每分鐘最多 10次 請求
    每月最多 1000次 請求

    流量計費：
    每分鐘最多 10,000次 請求     		每分鐘 60,000 Token
    每分鐘 500次 請求
    每天 10,000次 請求     		每分鐘 300,000 Token
    每分鐘 500次 請求
