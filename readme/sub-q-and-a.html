回首頁
  1. 這程式安全嗎
    此程式無需修改FF14遊戲檔案，且翻譯功能皆使用API請求，請放心使用。
  2. 這程式怎麼運作的
    此程式會擷取記憶體中的對話字幕，然後使用API請求翻譯，最後將翻譯結果顯示在程式中。若有勾選名詞校正，則會盡量修正FFXIV的專有名詞。
  3. 名詞沒有被正確翻譯
    此程式只會修正 對照表 內有的詞彙，若詞彙不在對照表裡將不會被程式修正，只能等對照表更新或是以【回報翻譯錯誤】功能來回報。
  4. 這程式支援什麼語言
    支援日文和英文，但翻譯以日文版為主。
  5. 能與漢化一起使用嗎
    可以，程式有略過漢化字幕的功能。
  6. 如果遊戲是日文，聊天頻道是英文，該如何設定
    至【設定】>【翻譯】>將【遊戲語言】設定成日文，【玩家頻道】設定成英文即可。
  7. 翻譯視窗無法維持在最上層
    請將FF14的顯示設定改成「無邊框全螢幕 / Borderless Window / ボーダレスウィンドウ」。
  8. 每一句都要手動選取翻譯嗎
    不需要，除了對話選項和使用者介面以外都可以自動翻譯，不能自動翻譯的部分也能使用【翻譯螢幕文字】功能來翻譯。
  9. 無法下載翻譯對照表
    若使用加速器或VPN時無法存取Github，請至Tataru Helper Node的【設定(齒輪按鈕)】>【系統設定】將SSL驗證關閉。
  10. 出現翻譯失敗的錯誤訊息
    若使用加速器或VPN時無法翻譯字幕，請至Tataru Helper Node的【設定(齒輪按鈕)】>【系統設定】將SSL驗證關閉。
  11. 啟動後顯示「這個作業系統不支援 .NET Framework 4.7.2」的訊息
    安裝 Microsoft .NET Framework 4.7.2
  12. 程式無法自動翻譯
    • 至 我的雲端 下載「fix.bat」，對其右鍵使用管理員身分執行，執行後按下任意鍵開始修復，修復完畢重新開機
    • 若無法下載fix.bat請在左下搜尋輸入「cmd」，找到「命令提示字元」對其滑鼠右鍵，點選「以系統管理員身分執行」，然後複製以下指令按下Enter執行
      secedit /configure /cfg %windir%\inf\defltbase.inf /db defltbase.sdb /verbose
  13. 如何翻譯對話選項和介面文字
    詳見【翻譯螢幕文字】說明頁
  14. 如何重新翻譯
    滑鼠點選程式的翻譯文字，再點選彈出視窗裡的【重新翻譯】按鈕即可
  15. 哪個翻譯引擎比較好
    目前推薦使用【ChatGPT】，其次為【有道翻譯】和【彩雲小譯】，但上述翻譯引擎都會有翻不好的時候，遇到這情況請嘗試使用其他翻譯引擎(點選程式的翻譯文字選擇其他翻譯引擎重新翻譯)
  16. 視窗半透明失效
    請將【NVIDIA控制面板】的【廣域設定】恢復成預設值(感謝avbkopye沒牙提供解法)
    或是使用【全新安裝】的方式重新安裝NVIDIA驅動程式(安裝時選擇自訂即可看到此選項)
  17. 要等很久才會顯示翻譯
    本應用程式是使用線上翻譯引擎， 因此翻譯速度取決於自身網路狀況和翻譯引擎方的伺服器狀況
  18. 出現「翻譯錯誤」
    原因是該翻譯引擎無法連線，或是遇到流量限制，只能等翻譯引擎解除限制，或是暫時使用別的翻譯引擎
  19. 如何移動視窗
    按住左上的 按鈕即可移動視窗
  20. 旁白字幕無法自動翻譯
    請安裝最新版本的Tataru Helper Node
  21. 無法套用變更後的設定
    更改設定後需點選右下角的【儲存變更】按鈕才會生效
  22. 按鈕有點擋畫面
    開啟【設定】視窗，將【隱藏按鈕】打勾
  23. 如何重置設定
    開啟【設定】視窗，點選【恢復預設】按鈕
  24. 如何查看程式錯誤訊息
    Ctrl+F12
  25. 如何更新Tataru Helper Node
    請至這裡下載最新版本
  26. 如何移除Tataru Helper Node
    至【新增或移除程式】移除Tataru Helper Node
  27. 如何查看錯誤回報的回覆
    請至這裡查看
